# Canonical Correlation Forests (CCFs)

## Prerequisites
1. Numpy
2. SciPy
3. Matplotlib

### Code
For classification example run the following command:
```bash
python3 classification_example.py
```

For regression example run the following command:
```bash
python3 regression_example.py
```
